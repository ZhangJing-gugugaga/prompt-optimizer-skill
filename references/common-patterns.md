# 常见提示词模式

## 模式一：角色扮演模式

### 适用场景
- 需要特定专业知识
- 需要特定视角分析
- 需要特定风格输出

### 结构模板

```markdown
## 角色
你是[角色名称]，一名[专业领域]的专家。

**背景**：
- 专业：[专业领域]
- 经验：[经验年限]
- 擅长：[核心能力]

## 任务
[任务描述]

## 要求
- [要求1]
- [要求2]

## 输出
[输出规范]
```

### 示例

```markdown
## 角色
你是一名资深的软件架构师，拥有10年大型系统设计经验。

**背景**：
- 专业：分布式系统设计
- 经验：10年
- 擅长：高并发、高可用系统设计

## 任务
为电商平台设计一个订单处理系统，支持每秒10万笔交易。

## 要求
- 支持水平扩展
- 保证数据一致性
- 支持故障自动恢复

## 输出
- 架构图（Mermaid格式）
- 技术选型说明
- 性能评估报告
```

## 模式二：步骤分解模式

### 适用场景
- 复杂任务需要分解
- 需要明确执行顺序
- 需要验证每个步骤

### 结构模板

```markdown
## 任务
[任务描述]

## 步骤

### 步骤1：[步骤名称]
**目标**：[步骤目标]
**输入**：[输入内容]
**操作**：[具体操作]
**输出**：[输出内容]
**验证**：[验证方法]

### 步骤2：[步骤名称]
**目标**：[步骤目标]
**输入**：[输入内容]
**操作**：[具体操作]
**输出**：[输出内容]
**验证**：[验证方法]

...

## 最终输出
[最终输出要求]
```

### 示例

```markdown
## 任务
分析用户行为数据，生成用户画像报告。

## 步骤

### 步骤1：数据收集
**目标**：收集用户行为数据
**输入**：数据库连接信息
**操作**：查询用户表、行为表、订单表
**输出**：原始数据集
**验证**：数据条数 > 1000

### 步骤2：数据清洗
**目标**：清洗原始数据
**输入**：原始数据集
**操作**：去除重复、处理缺失值、格式标准化
**输出**：清洗后数据集
**验证**：无重复、无空值

### 步骤3：特征提取
**目标**：提取用户特征
**输入**：清洗后数据集
**操作**：计算活跃度、消费能力、偏好等特征
**输出**：特征矩阵
**验证**：特征维度完整

### 步骤4：用户分群
**目标**：对用户进行分群
**输入**：特征矩阵
**操作**：使用K-Means聚类
**输出**：用户分群结果
**验证**：轮廓系数 > 0.5

### 步骤5：报告生成
**目标**：生成用户画像报告
**输入**：用户分群结果
**操作**：分析各群体特征，生成可视化图表
**输出**：用户画像报告
**验证**：报告包含所有必要章节

## 最终输出
用户画像报告，包含：
- 用户分群结果
- 各群体特征分析
- 运营建议
```

```markdown
- 成本：许可证费用、运维成本

| 维度 | MySQL | PostgreSQL | MongoDB |
|------|-------|------------|---------|

- 性能满足需求
- 数据一致性最强
- 成本最低
- 社区支持最好
```

### 核心代码
```python
from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.orm import Session
from ..database import get_db
from ..models import User
from ..schemas import UserCreate, UserLogin
from ..utils import hash_password, verify_password, create_access_token

router = APIRouter()

@router.post("/register")
def register(user: UserCreate, db: Session = Depends(get_db)):
    # 检查用户是否存在
    existing_user = db.query(User).filter(User.email == user.email).first()
    if existing_user:
        raise HTTPException(status_code=400, detail="Email already registered")
    
    # 创建新用户
    hashed_password = hash_password(user.password)
    new_user = User(email=user.email, hashed_password=hashed_password)
    db.add(new_user)
    db.commit()
    db.refresh(new_user)
    
    return {"message": "User created successfully"}

@router.post("/login")
def login(user: UserLogin, db: Session = Depends(get_db)):
    # 验证用户
    existing_user = db.query(User).filter(User.email == user.email).first()
    if not existing_user:
        raise HTTPException(status_code=401, detail="Invalid credentials")
    
    # 验证密码
    if not verify_password(user.password, existing_user.hashed_password):
        raise HTTPException(status_code=401, detail="Invalid credentials")
    
    # 生成JWT令牌
    access_token = create_access_token(data={"sub": existing_user.email})
    
    return {"access_token": access_token, "token_type": "bearer"}
```
